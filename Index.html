ğŸ“š Apa itu HTML?

HTML adalah singkatan dari HyperText Markup Language.

Penjelasan Sederhana:
HTML seperti kerangka rumah. Jika website adalah rumah, maka HTML adalah fondasi dan rangka yang menentukan struktur dasar halaman web.

HTML menggunakan "tag" untuk memberi tahu browser:

  • Mana yang judul, mana yang paragraf
  • Bagaimana cara menampilkan gambar
  • Cara membuat link ke halaman lain
  • Dan masih banyak lagi!

<!DOCTYPE html>

<html>
Tag pembuka dan penutup untuk seluruh dokumen HTML
<head>
Berisi informasi tentang halaman (tidak terlihat di browser)
<title>
Judul yang muncul di tab browser
<body>
Berisi semua konten yang terlihat di halaman web
